Voucher QC Tool

Automated quality control for e-commerce voucher codes

How It Works

1. UTM Link

Start with product URL with UTM parameters

→

2. Add to Cart

Automatically add product to cart

→

3. Apply Voucher

Enter and apply discount code

→

4. Verify

Check if voucher applied correctly

→

5. Report

Generate detailed QC report

Supported Voucher Types

  • Percentage Discount: e.g., 10% off entire order
  • Fixed Amount: e.g., $20 off entire order
  • Free Shipping: Removes shipping costs
  • BOGO (Buy One Get One): Buy one item, get another at a discount
  • Tiered Discounts: Spend more, save more
  • Combination: Multiple discount types combined

Key Features

  • Automated Testing: No manual checkout process needed
  • Parallel Testing: Test multiple vouchers simultaneously
  • Detailed Reports: HTML, JSON, CSV, Excel formats
  • Screenshots: Visual verification at each step
  • Configurable: Works with different store themes
  • UTM Support: Test marketing campaign tracking

Usage

# Initialize configuration
npm run init

# Update store configuration in config/stores.json

# Run tests
npm run run -- --file config/tests.json
